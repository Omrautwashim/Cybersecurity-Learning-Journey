## Day 6 – OSI Model Deep Dive

**What I Learned:**
- The OSI model has 7 layers.
- Each layer has a job—from physical cables to user applications.
- Helps in troubleshooting network problems.

**Layers:**
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application
